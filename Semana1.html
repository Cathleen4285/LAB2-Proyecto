Clase presencial

En esta clase conocimos a nuestro profesor Óscar Castillo Nevada, él estudió ingeniería mecatrónica, también tiene asesora marcas personales en productos digitales. También nos presentamos entre nosotros en una actividad dinámica. Cada uno mencionó su nombre, su proyecto de laboratorio 1, sus hobbies y nos divertíamos con las preguntas “rompe hielo” del profesor. Cada alumno destacó habilidades y debilidades, y algunos se comprometieron a actividades para colaborarnos entre nosotros. Reforzamos nuestros conocimientos en los conceptos, como: Innovación, originalidad, empatía, feedback y feedforward.

Clase virtual

En esta clase nos presentaron el sílabo del curso, que aprenderemos durante todo este proceso, además también nos fue mostrado herramientas y aparatos que podremos utilizar para nuestro proyecto de innovación. Nos presentaron las herramientas de Visual Studio Code con la que podremos configurar nuestra bitácora (en la que estás interactuando en este momento), para crearla, me cree una cuenta en Github, la cual trabajaremos al lado de Visual Studio Code y cada cambio que hiciera lo guardaba con Commit Chages.
